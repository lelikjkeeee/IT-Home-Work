создаем репозиторий в гите
прописываем git init
git add -A    -    добавляем все файлы (можно git add . или git add log.txt)
делаем коммит на наше действие git commit -m 'add: log.txt'
и пушим в гит хаб git push origin master (master - название ветки в которую пушим)


после каждого изменения файлов опять добавляем файлы - git add -A
делаем коммит git commit -m 'update: change log.txt'
и пушим git push origin master
